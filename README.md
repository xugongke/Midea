# 美的热水器电控项目

本仓库包含美的热水器两块电路板的固件源代码，使用 Git 进行版本管理。

## 项目结构

| 目录 | 板型 | MCU | 工具链 | 说明 |
|------|------|-----|--------|------|
| `hd1a00a8v4/` | 显示板 | SH79F326 (中颖) | Keil ?Vision | 按键触摸、LED 显示、EEPROM、I2C |
| `hd1a00b5_new_v4/` | 控制板 | STM8S005 (意法) | IAR EWSTM8 | 主控逻辑、NTC 采温、电机控制、UART |

各板内部均保留历史版本目录（`Source_v0~v15` / `APP_v0~v8` 等），以便追溯老工程师的迭代过程。

## 目录约定

- 源代码：`*.c` / `*.h`
- 工程文件：显示板 `Keil/*.uvproj`，控制板 `*.ewp` / `*.eww`
- 文档资料：`*.docx` / `*.xlsx` / `*.txt`
- 编译产物（`.o/.obj/.hex/.map/.lst` 等）已在 `.gitignore` 中忽略，请勿提交

## 开发说明

修改代码前请确认：
1. 显示板源码位于 `hd1a00a8v4/Source/`（历史版本在 `Source_vN/`）
2. 控制板源码位于 `hd1a00b5_new_v4/APP/`（历史版本在 `APP_vN/`）
3. 提交时请遵守提交规范，描述清楚改动点
