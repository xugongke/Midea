## 一、纯光伏模式防呆自动切换（★ 重点确认）

**触发条件（草案）**：水温 < 40℃ **且** 光伏直流电压 < 30V **且** 持续 2 小时温升 < 1℃，三者**同时**满足时，自动从"纯光伏模式"切换至"混动模式"。

**切换后行为（已与贵司确认）**：切换结果**掉电保存**，断电再上电仍保持混动模式；用户可手动切回纯光伏。

### 待确认问题

**Q1. 三个条件是否必须"同时"满足才切换？**
- [ ] 是，三者同时满足才切换（**我方推荐**，避免误触发）
- [ ] 否，任一条件满足即切换
- *理由：防呆逻辑应保守，避免因单一参数抖动导致频繁误切换。*

**Q2. "持续 2 小时温升 < 1℃" 的判定方式？**
- [ ] **方式A（固定窗口）**：进入纯光伏模式后记录起始水温，计时满 2 小时那一刻比较一次；温升 < 1℃ 即判定满足。
- [ ] **方式B（滑动窗口）**：持续监测，任意 2 小时窗口内温升 < 1℃ 即触发；若中途温升 ≥ 1℃，则计时清零重新开始。
- *我方建议：方式B。它更贴近"连续 2 小时未有效升温"的本意，能更早识别光伏供热不足；实现复杂度略高，但我方可实现。请贵司选择。*
